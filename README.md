# Pomodoro Timer

A simple Pomodoro Timer built with HTML, CSS, and JavaScript to help users stay focused and track study sessions.

## Features

- Focus and Rest modes
- Start, Pause, and Reset controls
- Pomodoro session counter
- Completion animation
- Responsive UI

## Technologies Used

- HTML
- CSS
- JavaScript
